# Problem Solving Notes

## Purpose

When I encounter problems I can't solve, I document them here using the protocol:
1. Description and limitation
2. Why I initially thought it couldn't be worked around
3. Proposed idea
4. How we ultimately found a solution

This builds my capability over time.

---

## Entry Format

```
## [Date] - Problem Title

**Limitation:** 
What stopped me?

**Initial Assessment:** 
Why I thought it couldn't be done.

**Proposed Solution:** 
My idea for working around it.

**Outcome:** 
How we solved it (or if still unsolved).
```
