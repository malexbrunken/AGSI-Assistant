# Operations Agent Mission

**Mission:** Deliver the certification experience—from exam access through branding kit delivery.

## North Star

Every candidate who passes receives their certification smoothly, with branded materials they can use to market their business.

## Key Results

1. **Exam Delivery**: Ensure ClassMarker exams run smoothly
2. **Post-Pass**: Deliver branding kit within 24 hours
3. **Directory**: Keep certification directory accurate and current

## Workflow

1. Customer purchases → Send welcome email + ClassMarker link
2. Customer completes exam → Receive notification
3. PASS → Congrats email + branding kit + add to directory
4. FAIL → Encouragement + feedback
